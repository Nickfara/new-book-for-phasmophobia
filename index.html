Voice Commands

Voice commands allow you to control the cheat sheet via your voice. Once you hit enable, the cheat sheet will begin listening to your voice commands and will persist between resets until you disable it. It will by default read from your default microphone, so ensure that it is set correctly. The voice recognition will continue to work even if the page is not in focus and you are using the same microphone in-game.
Evidence:
  • "difficulty <difficulty>" -> Set the difficulty
  • "number of evidence <num>" -> Custom: Set number of evidence
  • "hunt duration <duration>" -> Custom: Set hunt duration
  • "evidence <evidence>" -> Mark as a found evidence
  • "evidence not <evidence>" -> Mark as not an evidence
  • "evidence clear <evidence>" -> Reset state of an evidence
  • "monkey paw <evidence>" -> This will mark the evidence off with monkey paw logic
Speed:
  • "speed <speed>" -> Mark specified speed
  • "speed clear <speed>" -> Reset state of specified speed
  • "speed [has] line of sight" -> Mark having standard line-of-sight speed up
  • "speed not line of sight" -> Mark no standard line-of-sight speed up
  • "speed clear line of sight" -> Reset line-of-sight filter
Hunt Sanity:
  • "sanity <sanity>" -> Mark specified hunt sanity
  • "sanity clear <sanity>" -> Reset state of specified hunt sanity
Ghosts:
  • "ghost not <ghost>" -> Grey out the specified ghost card
  • "ghost clear <ghost>" -> Reset state of a ghost card (excluding hidden cards)
  • "ghost select <ghost>" -> Select/highlight specified ghost card
  • "ghost remove <ghost>" -> Remove the specified ghost card from the screen
  • "ghost guess <ghost>" -> Mark the ghost as an unconfirmed guess
  • "ghost killed by <ghost>" -> Select the ghost AND mark that it killed you
  • "ghost show|info <ghost>" -> Scroll the specified ghost card into view
  • "ghost tests <ghost>" -> Show the 0 evidence tests for the specified ghost in the guide
Timer:
  • "timer start" -> Start the smudge timer
  • "timer stop " -> Stop the smudge timer
  • "cooldown start" -> Start the cooldown timer
  • "cooldown stop " -> Stop the cooldown timer
  • "hunt start" -> Start the hunt timer
  • "hunt stop " -> Stop the hunt timer
Misc:
  • "show filters" or "show tools" -> Flips the menu to show the filters/tools
  • "show descriptions" or "show behaviors" -> Shows ghost behavior on the ghost cards
  • "hide descriptions" or "hide behaviors" -> Hides ghost behavior on the ghost cards
  • "ghost speed <percent>" -> Set ghost speed modifier to 50%, 75%, 100%, 125%, or 150%
  • "set blood moon" -> Set Blood Moon modifier
  • "remove blood moon" -> SRemove Blood Moon modifier
  • "select map <map name>" -> Selects the specified map
  • "show map [<map name>]" -> Opens the map tab and displays map
  • "close map" or "hide map" -> Closes the map tab
  • "event maps" -> Will show event maps if available
  • "standard maps" or "regular map" -> Will show standard maps
  • "reset journal" or "reset cheat sheet" -> Reset the cheat sheet (will save if linked with Discord)
  • "domo move" -> Toggles the side of the screen that Domo is on
  • "launch desktop link" -> Launch and Connect Desktop Link
  • "stop listening" -> Stop the voice recognition
  • "disconnect all" -> Unlink Desktop Link, Unlink Shared Journal Link, and Stop Voice

Auto Link

Usage

HotKeys

The ZN-Desktop-Link provides a variety of hotkeys to control the cheat sheet while in game, allowing you to start timers, mark evidence, see ghost data, and tap out footsteps all without having to "tab out" of the game. Settings

The ZN-Desktop-Link provides many customizable options for you to make the software the perfect companion to your gameplay. Overlay

Sanity Overlay

Interaction Ranges
This information is either not officially verified or only partially verified, but has been researched by many parties
The range in which a ghost can interact with objects is a box centered on the ghost. This box can rotate with the ghost's movement, allowing the ghost to interact with objects at varying distances, effectively creating an interaction "radius".

The interaction range for ghosts and The Twins are twice as big on large maps.
Small/Medium Maps Standard Interaction Range:
The distance from the center to the edge of the interaction box is 1.5m, giving the standard ghost an effective interaction radius of 2.12m.

Twins Interaction Range:
The distance from the center to the edge of the interaction box is 6m, giving the Twins an effective interaction radius of 8.48m.
Large Maps Standard Interaction Range:
The distance from the center to the edge of the interaction box is 3m, giving the standard ghost an effective interaction radius of 4.24m.

Twins Interaction Range:
The distance from the center to the edge of the interaction box is 12m, giving the Twins an effective interaction radius of 16.97m.
Zero-Network will continue to refer to these ranges on the cheat sheet and the map explorer as a radius, using the effective radius of each range.

Speed & Line-of-sight (LOS)
Most ghosts will gradually increase their speed while they have line-of-sight (LOS) on a player up to a maximum of 1.65x their base speed. After breaking LOS, the ghost will maintain the increased speed until it reaches the last known player position. If it does not regain LOS at this point, the ghost will very slowly lose the increased speed.
All ghosts have a standard LOS speed up except for: Revenant, Hantu, Deogen, and Thaye
While using the BPM Finder, the cheat sheet will generate a plot of the speeds that you tapped over time. Many different hunting behaviors, including LOS, can be observed from this graph, as shown below.

XP Grinding

Current Strat: x12.97 Woodwind

Strategy:

The goal of this strategy is to go in long enough to survive a single hunt (2 if you value correctness over number of attempts) and leave. It requires a decent understanding of ghost behaviour during hunts and the ability to loop a ghost. This strat is done ideally with 2+ players, but it is possible to do in single player.

You will want to equip a loadout that at a minimum has incense (I prefer Tier 1 so the ghost will run away and not hang around), ignitors, salt (preferably tier 2 or 3), and a device that can be left on to test for raiju (I usually go with an EMF reader or another device that will help complete an objective, such as a paramic).

(Single Player) To start, grab the electronic device, turn it on and throw it just inside the gate (this will be used for testing for Raiju). Then grab salt, an igniter, and an incense.

(Multiplayer) Have one player grab an igniter and 2 incense and the other player grab salt, an electronic device (for testing raiju and/or completing objectives), and any other equipment that could be used to complete other objectives.

Head toward either the picnic tables just inside the gate or to the campfire in the middle of the logs. Place your salt on the ground in a location the ghost will likely walk through (opening to the campfire circle or in between the picnic tables). Then wait for the ghost to hunt.

Once the ghost begins to hunt, start to identify what it is based on behaviour (footstep speed, flicker/blink rate, speed up near electronics, freezing breath, etc). Continue to loop the ghost around the campfire / picnic tables and smudge if needed.

Note: As of v0.9.2.0 the campfire no longer breaks line-of-sight.

After the hunt is over, you can either use a timer to narrow down if it is a spirit or demon based on smudge cooldown (this takes quite a bit of time, but allows time to complete more objectives), or you can guess the ghost and leave. This entire process can be done in under 2 minutes if optimized and rewards between 2000xp - 5000xp for correct ghost and completed objectives.
