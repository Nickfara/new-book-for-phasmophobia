Welcome to the
Zero-Network Mobile Link!

This app is designed to be a companion to the Unofficial Phasmo Cheat Sheet's Shared Journal Link for console players or players that cannot run the ZN Desktop Link on their computer.

Here is how to use the ZN Mobile Link:

  1. At least one person in your group must be using the full version of the Unofficial Phasmo Cheat Sheet. Have them create a Shared Journal Link and share the code with you.
  2. Enter that code into the ZN Mobile Link and join the room
  3. Enjoy the Features!
    • Evidence, Ghosts, and Timers are synced between everyone in the same room (you can mark evidence and start/stop timers too!)
    • Click on the list of ghosts to view all the data about that ghost (its the same as what is on the ghost cards on the Cheat Sheet)
    • Use the text chat to communicate with other members in your room that are also using the ZN Mobile Link or the ZN Desktop Link on PC

If you experience any bugs, please let us know on our Discord!
